Passos para Hospedar Um Projeto no GitHub Pages

  1. Crie um repositório no GitHub com o mesmo nome do seu projeto.
  2. No terminal, incialize o repositório local com git.init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commit com git commit -m "first commit"
  5. Conecte ao repositório remoto com:
    git remote add prigin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main .
  7. No GitHub, vá em Settings > Pages.
  8. Em "Branch", selecione main e a pasta / (root) . Clique em save
  9. Após alguns segundos, seu site estará disponível no link exibidos em "Pages".
